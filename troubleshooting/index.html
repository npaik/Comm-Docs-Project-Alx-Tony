Troubleshooting

This section will focus on the most common issues that might occur while following the provided documentation. You can look up the error codes, and if it matches yours, check the possible issue and solution.

Error Probable cause Solution
Windows: unable to access C:/Users/UserName/.gitconfig: Permission denied
Linux, MacOS: could not lock config file /Users/UserName/.gitconfig: Permission denied		 You ran git config --global init.defaultBranch "branch name" and Git is unable to modify the global configuration file located at ~/.gitconfig due to permission issues Try running the command as an administrator or ensure that you have sufficient permissions to modify the .gitconfig file
aborting commit due to empty commit message You are trying to commit without specifying a commit message Run git commit -m "your commit message" to specify a commit message
failed to push some refs to "remote-repo-url" hint: Updates were rejected because the remote contains work that you do not have locally Remote repository has been updated since your last pull or fetch, and the changes conflict with your local changes. You can run git pull to fetch the latest changes and merge them with your local changes, and then try pushing again.
your local changes to the following files would be overwritten by merge You are running git pull command, and your local changes conflicting with changes in the remote repository Commit your local changes before running git pull, or stash your changes using git stash and then run git pull
the current branch master has no upstream branch You are trying to push changes to a branch that doesn't have an upstream branch set You can use the -u flag when pushing to set the upstream branch, like git push -u origin master
automatic merge failed; fix conflicts and then commit the result You are trying to merge two branches with git merge "branch name" and there are files with conflicts that haven't been resolved yet You can resolve the conflicts by editing the files or using a merge tool, and then commit the changes to complete the merge
